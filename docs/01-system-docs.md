Nome do projeto: MindJournal AI

Objetivo:
Sistema de diário pessoal inteligente com chat, histórico de conversas, upload de arquivos TXT/PDF e preparação para integração futura com IA.

Funcionalidades:
- Enviar mensagens pelo chat
- Receber respostas textuais simuladas pelo back-end
- Criar e recuperar sessões de conversa
- Listar histórico por sessão
- Fazer upload de arquivos TXT e PDF
- Exibir progresso de upload
- Verificar saúde da API pelo endpoint /api/health

Requisitos mínimos:
- React no front-end
- Spring Boot no back-end
- Dois repositórios separados
- README.md em cada repositório
- AGENTS.md em cada repositório
- Arquitetura separada por responsabilidades
